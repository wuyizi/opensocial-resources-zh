Class BaseType
 
Packageorg.opensocial.client.base
Classpublic class BaseType
SubclassesCollection, DataRequest, DataType, Environment, GadgetsIo, GadgetsViews, Globals, ResponseItem

The base type interface for most of opensocial related types in this package.

This is the root class for some reflection usage except the ConstType and ArrayType series.
