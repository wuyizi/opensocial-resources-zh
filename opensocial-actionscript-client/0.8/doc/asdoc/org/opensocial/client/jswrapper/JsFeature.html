Packageorg.opensocial.client.jswrapper
Classpublic class JsFeature
SubclassesGadgetsIo, GadgetsRpc, GadgetsViews, GadgetsWindow, Opensocial

Base class for all features wrapped from Javascript. Core and abstract functionalities are defined in this class, and other feature related codes are in sub classes. The client will load several instances of JsFeature to support the whole API.
