handleComplete()method 
protected function handleComplete(rawData:*):void

The function for the async callback which is registered on the client callback manager. It takes in the async response raw data to create a corresponding event then dispatches it. This function will be override by derieved classes.

Parameters
rawData:* — The raw response object.
send()method 
public function send(client:OpenSocialClient):void

Sends out the request using client's async call api. If this request instance is running, then this method will do nothing.

The underlying client has no knowledge on the event-driven system.

Parameters
client:OpenSocialClient — The client, could be JsWrapperClient or RestfulClient.
