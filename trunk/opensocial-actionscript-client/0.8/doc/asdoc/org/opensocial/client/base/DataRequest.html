Classpublic class DataRequest
InheritanceDataRequest BaseType

Wrapper of opensocial.DataRequest object in javascript.

This class is just used as a namespace for these constants. Different from the javascript API, the data request real logic are implemented seperately in org.opensocial.client.jswrapper.OpenSocial feature class and restful client class (not exist yet).

TODO In future, batch fetch should be supported. So this class needs to be expanded.

public static function newPeopleRequestParams(first:int = 0, max:int = 20, details:Array = null, sortOrder:String = null, filter:String = null):Object

Prepare a people request parameter object. Keys are defined in DataRequest.PeopleRequestFields.

Parameters
first:int (default = 0) — The value for DataRequest.PeopleRequestFields.FIRST, default is 0.

details:Array (default = null) — The value for DataRequest.PeopleRequestFields.PROFILE_DETAILST, which is an array of Person.Field. The default in javascript api are the person's id, name and thumbnail url. So no need to pass these three fields here.
