Library: opensocial-flashapi.js
Overview
A wrapper for opensocial javascript API into actionscript.

It works with the Opensocial Flash API. In this file, the local var with underscore indicates that this obj is a JSON object wappered from opensocial data types.

flashAPI.isAPIReady(objectId)
Indicates if the opensocial library is loaded. Initialize the opensocial and gadgets namespaces. Initialize the flashAPI.swfObj element. The objectId of the swf is defined in gagdet spec. If the swf is embeded by calling flashAPI.embededFlash function, the detault objectId will be 'flashAPI.swfObj'. This function is called by the swf's ExternalInterface continuingly until it returns true.
parameters
string? objectId The object Id passed from the swf.
returns
boolean true if all javascripts are loaded.
