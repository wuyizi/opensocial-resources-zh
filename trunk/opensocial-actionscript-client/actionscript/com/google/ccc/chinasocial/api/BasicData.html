Packagecom.google.ccc.chinasocial.api
Classpublic class BasicData

Initial data set for an app.

This set is a composition of some data types, including: Viewer(Person), Owner(Person), Viewer Friends(Collection of Person), Owner Friends(Collection of Person), Viewer AppData(Key-Value) and Owner AppData(Key-Value).
